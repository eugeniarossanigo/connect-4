### Connect 4
Strategy game

### Installation
```bash
npm i
```

### Usage
```bash
npm run dev
```

### Instructions
* Players must connect 4 of the same colored discs in a row to win.
* Only one piece is played at a time.
* Players can be on the offensive or defensive.
* The game ends when there is a 4-in-a-row or a stalemate.
